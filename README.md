# Awesome Claw Assets

A curated collection of high-signal OpenClaw assets: skills, registries, curated libraries, tooling, and ecosystem resources.

The goal of this repo is simple: keep the good stuff, cut the sludge.

OpenClaw's ecosystem is growing fast. Useful skills exist across ClawHub, GitHub, niche domain libraries, and broader SKILL.md-compatible agent ecosystems. This repo is meant to become a public map of that landscape.

## What belongs here

- OpenClaw-specific skill registries and archives
- Curated awesome lists and skill collections
- Domain-specific skill packs worth studying or reusing
- Tooling for writing, validating, linting, packaging, or distributing skills
- Ecosystem resources that help people discover, trust, and use Claw assets

## What does not belong here

- Blind mirrors of every public skill
- Spam farms and low-effort “AI does everything” junk
- Obvious growth-hack bait
- Anything unsafe, shady, or impossible to verify at a glance

## Security note

Skills are not harmless markdown decorations.
They encode behavior.
Treat them like code.

Before installing or recommending a public skill:
- inspect the source
- check what tools it expects
- look for hidden scripts or unsafe defaults
- assume nothing is safe just because it is public
